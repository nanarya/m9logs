念願のニーハン。CB223Sを買った。

Jun 9, 2019 • kidm9

とにゃ～、キッドです！

今日はバイクを買った話。

前々から乗りたいとは思ってて、随分前に免許も取ってたんですが、ようやくタイミングが回ってきたので。

それなのに、買った途端に梅雨入りして乗れないのでのんびり記事でも書くことにしました。

HONDA CB223S

まずは買ったバイクの紹介から。

https://twitter.com/kidm9/status/1135006934975692800

https://twitter.com/kidm9/status/1135023604809121792

https://www.honda.co.jp/CB223S/

 

ちょっとクラシカルなデザインのバイクです。

写真になかなか出ないんですが、赤いタンクが結構良い色してるんですよね。

購入直前まで迷ってたんですが、CB400SSやSR400なんかも候補でした。

いずれも似たようなタイプのバイクですね。

CB223Sは2016年に生産終了になっているので、新品は手に入らないですが、中古ならまだ状態が良いものがそこそこあるみたいです。

排気量は223cc。ざっくりいわゆるニーハンのくくりになるクラスです。

今までの愛車がCD50で50cc（ボアアップして88cc）だったので、ずいぶん車体も大きくなりました。

http://m9logs.net/goingout/677

ファーストインプレッション

まだ数えるほどしか乗れてないですが、簡単に。

CD50からの乗り換えなので、そのあたりも念頭に置きながら読んでください。

 

まずはとにかくパワーがある。加速力やトルク感はやっぱり圧倒的に変わりました。

あと車体が大きくなった分、まだちょっと扱いきれていない部分も。

 

スピードがー、パワーがー、とネットには書いてあったりもしますが、街乗りしてる分には十二分ですね。

一方スピードを出していくと、80km/hまでは難なく走れますが、100km/hを超えたあたりで振動が大きくなってコントロールが難しくなってきます。

多分フルスロットルのアクセル全開で120km/h出ないくらいじゃないでしょうかね。（やってないのでわからないですが）

 

重量も、このクラスにしては軽い135kgなので、風に煽られたりするのもちょっと怖いところがあります。

高速でぶん回すようなバイクではないですね。

その分取り回しについても流石に原付ほどとはいかないですが、割と扱いやすいように思います。

162cmの短足な自分が乗っても両足のつま先はつくので、脚付きも悪くないですね。

 

あとはエンジンの音が気持ちいいですね。

ホンダの4ストローク単気筒エンジンは実家のような安心感です。

黒いマフラーは世間ではダサいという評判だそうですが、自分は割と気に入ってます。

 

あと案外良かったのは燃費。ちょっと急発進とかもしてみたりして、色々試したりしてたんですが、それでも37km/L出たので、普通に乗れば40km/Lくらい期待できるんじゃないですかね？

タンクが11Lなので、400kmくらいは給油なしで走ってくれそうです。

 

色々道具を揃えた

バイクに乗るには究極ヘルメットだけあればなんとかなるんですが、仮にも高速道路に乗れる乗り物なので、今までよりはしっかり装備を整えようと決心して、色々買いました。

ヘルメットはARAIのやつが2りんかんで安かったのでそれを購入。

安いと言っても数万円しましたが、安全第一です…！

https://twitter.com/kidm9/status/1135154061047582725

グローブ、雨用ウェア、シューズあたりはみんな大好きワークマンで購入。やっぱりコスパ素晴らしいですね。

プロテクター類やその他細々したものはamazonでポチっと。

 

安全運転コミネマンです。

ジャケットは胸肩肘に硬いプロテクターが入ってて、かつ街にも繰り出せそうなデザインなのでとても気に入ってます。ハーフメッシュなので、走り出すとめちゃくちゃ涼しいです。

同じモデルが2りんかんからOEMで出てるので、サイズ感見ておいたほうが良いと思います。L買おうか迷ったけどMで十分だったので。

 

あとはキャリアを楽天、サイドバッグサポートをamazonで買いました。

https://twitter.com/kidm9/status/1135057120540237824

流石にこれでキャンプいくのは厳しいのでｗ

https://twitter.com/kidm9/status/1137183885182230528

こんな感じ。

横の耳はダンデム用グリップと書いてありましたが都合よく荷物置ける面積が増えたのでグッジョブです。

一応耐荷重6kgと書いてありましたが、割とフレームにガッチリ固定できるタイプだったので、安心して使えそうです。

キタコのサイドバッグサポートと、エンデュランスのリアキャリア。一箇所固定用の穴共有しますが、干渉もなく普通に取り付けできました。

サイドバッグはまだ買ってないですが、荷物の積載にも余裕ができて、今からキャンプに行くのが楽しみです。

ツーリングをしてみたい

今まで原付だったからというのもあるんですが、なかなかツーリングという気分にもならなかったんですがせっかく良いバイクを買ったので、バイクに乗ることをメインに据えたツーリングとかもしてみたいですね。

もちろん、キャンプに釣りにカメラにと、相性の良い趣味は色々あるのでそれらも上手いこと組み合わせながら。

高速にも乗れるようになったので、遠出も積極的にしたいですね。

週末の楽しみがまた1つ増えました。
