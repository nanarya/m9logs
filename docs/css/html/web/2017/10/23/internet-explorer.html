Internet Explorerとその子孫

Oct 23, 2017 • kidm9

とにゃ～、キッドです。

最近更新してないなーと思ったのでのんびり更新。

今日はWEBの話です。

WEB制作者の悩みというと、いつも笑い話にもなる「IEが良くない」という話。

今までそんなに悩んだことが無かったんですが

（そもそも趣味サイトなので上手く表示できないページは捨て置けばいいやと考えてた）

最近新しいWEBページを制作していて、IEはともかくその子孫のEdge（Win10標準ブラウザ）には対応しとかないとなとなったわけです。

ただ、ちょっと頑張っていろんなタグやらスタイルやら使って作ってしまったもので、

Chromeでは理想通り動くけれど、Edgeでは未対応みたいなコードを使ってしまったらしい。

今悩んでいるところとしては、CSSでobject-fitを使っていたけど、Edge未対応なのでどうしようかなというところ。

imgタグにobject-fitを使って、div（大きさ可変）いっぱいに表示した画像を、アスペクト比変えずに表示したかったんだけど、

それができないとなるとimgじゃなくてbackground-imageで指定する感じになるのかな…？

もしくは固定サイズにする？悩ましいところです。

HTML5とかCSS3とか、規格がしっかり決まっているのに、対応されてないブラウザがあるって、どうなのよ。

とか思ってしまう今日この頃でした。

 

ちなみに最近はWordpressを使ったサイトを作ってます。

そのうちこのサイトも丸々デザインしたいな。

あと、Jekyllが気になっている。そのうち触ってみたいな～。
