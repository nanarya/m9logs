自己紹介：テレビゲームの話

Jul 18, 2017 • kidm9

ゲームが好きです。テレビゲームもアナログゲームもどっちも好きです。どっちも紹介したいけど、まずは馴染みの深そうなテレビゲームから。

いつから？どんなの？

昔からゲームが好きで、幼いころからよく遊んでました。任天堂とソニーの据え置き/携帯ゲームが多かったですかね。あとはPCゲームとかもやりますよ。今はネトゲとPS4が中心ですかね。でもGBASPとか引っぱり出したりもします。

FINAL FANTASY

好きなゲームといえば、なんといってもFINAL FANTASY！

ナンバリングは概ねプレイして、1番好きだったのは初めてのFFでもあったFF7！他にも好きなタイトルはありますが、1番好きなFFは？と聞かれたら迷わずFF7と答えます！

今は、枠がちょっと別になるけど、オンラインゲームのFF14にハマってます！メインは白魔道士のララフェルです！やってる人いたら仲良くしてください。

あとは、今実はFF11をやってみたい。FFって今1から15まで出てるんですが、その中で11と14がオンラインゲームなんです。で、今その14にハマっている。で、11はというと、やったことがないんです。PS2をネットにつないでプレイするような時代だったので、当時はできなかったんですね。なので、シナリオも出そろって、プレイする環境もある今、やってみるのもありなんじゃないかな～、と。

まあ、FFの話は書き始めると記事が埋まっちゃうので、このへんにしときまして。

ポケモン

またビッグタイトルだけど、ポケモンも好きです。しばらく休んでたけど、サンムーンで久々に復帰。新しい雰囲気に最初は戸惑いつつも、やっぱり面白かったです。プレイスタイルとしては、のんびり作りたい型のポケモンを作って、友達とバトルする感じです。なかなか独創的で面白い型もあるのでそのうち紹介したいなー。

好きなポケモンはラプラスです。そこそこ強くてそこそこ器用。でも強ポケモンにはなり切れない。なかなか難しいポジションだけど、そこを工夫していろいろ無理やり使うのが楽しい！他のポケモンも、なぜか水タイプが自然と多くなる傾向…。大体見た目で選んで、型を考えて作ってます。作るだけ作ってバトルはあんまりできてない。

そのほか

好きなゲームというとあれもこれもとなるけど、やっぱりこの2タイトルが自分の中を占める割合が大きいかな。あとは、例えば最近面白かったのといえば、ペルソナ5とか。独特の雰囲気がとても良く出来てて好きでした。ほかには、そろそろドラクエ11が発売するので、それも楽しみかなー。はやくリアルタイムでプレイしてるゲームの記事とか書きたいですね。

さて、実は好きなゲームの話といいつつ、1番好きなゲームを紹介してなかったりします。もったいぶる話でもないんですが、なんとなく気が向いた時に記事にしたいと思うので、今は秘密ということで。
