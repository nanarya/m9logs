KINGDOM HEARTS 3

Feb 11, 2019 • kidm9

とにゃ〜！キッドです。

先日、ついに出ましたキングダムハーツ3。

早速発売日にDLで最速入手してプレイしましたので、その感想とかもろもろ。

待ちに待ったナンバリング

KH1が2002年、KH2が2005年。間に外伝はいくつかあるにしろ、そこから14年。

随分待ったなぁ。

間の外伝は実はほとんど未プレイなので、随分長かったように感じます。

正直、発売前までは一応プレイしておこうかくらいの軽めのノリだったんですが、1週間前くらいから大分気になり出して、結果発売日からどっぷりでした。

https://twitter.com/kid_nanarya/status/1088451209390780423

発売前からwktk待機です。

そして…

https://twitter.com/kid_nanarya/status/1089354247748038656

発売日の2日後の朝にはクリアしている始末。なんだかんだ10年以上待ってたので、目一杯やりたかったんですよ。

ひたすらにやりつづける日々

発売日から、エンディングを見てもまだまだひたすらにやり続けます。

（というか先に進みたすぎてなかなかのんびりサブイベントやミニゲームを見られてなかったので、ゆっくりやりこみといった感じです。）

宝箱探しと幸運のマーク探しをしながら、ミニゲームもちょこちょこと遊びます。

今作、1ワールドだけやたら気合いが入ったワールドがあるので、ぜひ1回は攻略情報見ずに探索してみてくださいね。

そんなこんなで2週間。

https://twitter.com/kid_nanarya/status/1092052469427798016

ひたすらにやりつづけて、トロフィーコンプリート。

と言っても最後の1日は、埋めるためにちょっと攻略情報に頼っちゃったんですけどね。

隠しボスはレベル最高まで上げて、アルテマウエポン作って、エリクサー5個装備して、最強装備の状態で望んだんですが、楽勝でしたね…。そもそも物語の進行上で止まるのが嫌だったのでスタンダードで遊んでいたんでした。

そんなこんなで久々にトロフィーコンプリート。テイルズ・オブ・ヴェスペリア（PS3）以来2作目とかかな？あんまりトロフィーに興味ないので、というか面倒な作業が苦手なのでやらないことの方が多いんですよね。今作はめずらしく頑張りました。

物語としてのキングダムハーツ

物語は、やはり途中が抜けている分、若干ついていけない感じはありましたが、2のときから大分ややこしかったので、オリジナル部分の話はふわっと理解していればいいのかな、という感じで進めました。

もうさすがに賢者アンセムとゼアノートとゼムナスとアンセムがいてハートレスとノーバディとアンヴァースと、キーブレードとχブレードと、なんだかんだとややこしすぎますが、まああんまり難しく考えないことにしました。

ディズニーパートはいつもくらいのボリュームで、ダイジェストになった物語にソラたちが少し絡んで、物語が進む形式。物語の舞台を実際に歩けるのは、ファンなら垂涎ものなのだろうな、とか思いながら、特に思い入れの無い自分でも案外わくわくしながら散策しました。物語の世界に入れるあの感じ、好きなんですよね。

そして最後はお決まりの隠しムービーがあるんですが、これが賛否両論らしい。個人的には案外面白いメッセージだったんじゃないかな？と思うのでした。真相はどうだかはわかりませんけどね。気になる方はこのへんの考察をどうぞ。

https://ga-m.com/n/kingdom-hearts-3-secret-imi-kaisetu/

 

ディズニーという最高のコンテンツ

正直、ディズニーがそれほど好きというわけではないんですよね。もちろん嫌いじゃないんですが、特別強い思い入れもなく。

キングダムハーツに出てくるから、ある程度作品を知ってるくらいだったんですよ。

でもキングダムハーツを通して知った作品が、案外好きだったりすることもあって。ナイトメア・ビフォア・クリスマスとか。

今作は新しい作品も多かったので、見たことないものも多かったんですが、その中でもアナ雪はすごく面白くて。というかオラフのキャラクターが大好きで。

ということでAmazonプライムビデオで見てみたんですよ。

そしたらものすごく面白くて。思わず続編も見ちゃったし、なんなら本編も続編も2周しました。

続編は短編で20分くらいのアニメーションなんですが、とにかくオラフがかわいくて。あと、物語がとてもよくまとまっていて。思わず泣いちゃいました。本編でもそんなことなかったのに。

まあ何が言いたいかというと、アナ雪はいいぞ。

それと、ベイマックス。

これも公開当時に話題になっていたからそのうち見たいと思ってたんですが、今作を通して世界観が知れたので、入りやすいかなと思って、アナ雪続編を見た勢いで続けて見ました。

結果、号泣。

オラフで泣いて、ベイマックスで泣いて、その日は大変な1日でした。

普段映画はあまり見ない方なんですが、たまにはこういう休日もいいな、と思いました。

クリアした後に（というかさっき）見たので、ゲーム本編よりこっちに熱が入っちゃいましたね。閑話休題。

次回作が楽しみですね

次回作は何十年後に出るんですかね？＾＾

まあ、きっと10年でも20年でも待ってるんだろうなと思います。

というか野村さんはじめ開発陣の年齢がやばくなるから早く作って欲しいですね（

またソラたちと、ディズニーの世界を飛び回る日々が来ることを楽しみに待つことにしましょう。
