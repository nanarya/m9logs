Amazon探したら15しか置いてなかったんですが、これの13です。

Peak Design / Everyday Messenger 13

普段使いもできるメッセンジャーバッグです。

適度なサイズで収納力もあり、カメラをきっちりガードできる優れもの。

もちろん細かいポケットの作り等も撮影のストレスにならないよう考えられてます。

このPeak Designさんというのが、バッグのほかにもなかなかよく考えられた良いものを作ってまして、

【国内正規品】PeakDesign ピークデザイン キャプチャーカメラクリップ CCC-2.0
ピークデザイン

これが今欲しい！

カメラをホルスターみたいに「ガシャコンッ！」とハメられるものです。

バッグの写真を見ると横につかめそうなところがありますが、そこにもくっついたりします。

Peak Designさんはこういった、ほかのアイテムとの連携も上手なところです。

最近これの新型が出たとのことなので、国内流通始まったら買ってみようかと思ったりしているのでした。

α7II + Loxia 2/35はお散歩にもピッタリ

さて、カメラの話に戻りますが、新しいカメラはフルサイズ機としてはボディもレンズも小振りなので、取り回しの良さが気に入ってます。

新しいカメラは、以前のNEX-6と比べるとやはり重いですが、おさまりの良いサイズのおかげでそれほど気にはなっていない感じです。

新しいものを手にした満足感と、億劫にならない重量とのバランスが良かったのか、外に出る機会が増えた気がします。

最近はflickrの更新もかなり頻繁になってきてるので、よかったらそっちものぞいてみてください。

最近は自分で気に入る写真も撮れるようになってきて、段々上手くなってるのを感じます。スキルアップは楽しいですね。

 

flickrはこちら

 

Loxia 2/35、良いレンズです

コンパクトで、寄れて、フォーカスが気持ち良い。

35ｍｍの画角なので、広めに使えてテーブルフォトもできるし、一歩寄ればスナップでも普通の標準のような使い方もできる。

30cmくらいまで寄れるので被写体との距離に困ることもそれほど無い。

玉ボケ、光芒、フレアなども、意図して入れるととてもきれいに出てくれるので、いろいろ遊べます。

解放だとふわっとしていて、ノスタルジックな絵が出てくるので、ついつい解放で使いたくなりますが、

絞ると今度はかなり解像してくるので、一粒で二度おいしいレンズです。

最初の一本、良いものを買ってしばらく一本で頑張ろうと思って買ってみましたが、どうやら正解だったみたいです。

望遠ができない（当然ですが）以外は特に不便もなく、むしろ満足して使ってます。

写真を撮るとなんとなく雰囲気で絵っぽくなる感じのレンズなので、単純に使ってて楽しいですね。

最近はこんなところに行きました

まあflickrのアルバム見てもらうのが一番ではあるんですがｗ

お気に入りの写真を交えながら。

小石川後楽園

紅葉がとてもきれいでした。写真が上手な友人ができて、初めて一緒に撮影に行ったところ、腕の違いを思い知らされましたｗ

ただ、とても勉強になることが多く、一日でかなり上達した気がします。

浅草

とにかくすごい色で撮れた。これ、ほぼ撮って出しです。ほんとにこんな色してました。すげぇ。被写体パワーってやつですね。

江ノ島

なんとなく行った江ノ島、ライトアップがきれいでした。黒い猫と仲良くなりました。

等々力渓谷

またもや撮影スポットへ。小石川後楽園で紅葉は満喫できたので、この日は光で遊んだりしてました。

落ち葉で焼き芋を焼いていたりしたので、掃除を手伝っていただいたりした一幕も。秋を感じた一日でした。

山手西洋館

やってみたかった室内撮り。今は世界のクリスマスの展示がしてありました。人工物撮るの楽しい。

 

こんな感じで、あちらこちらで撮影してきました。

他にも近所の散策したり、川越行ったり、とにかくカメラを持ってうろうろ。

他の写真も見たい人は、flickrをのぞいてみてください。
