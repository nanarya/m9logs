都会の喧騒に疲れてしまったキッドさんは、このごろ漠然と、田舎に住みたいな〜と思うことがよくあります。

ただ、よし、田舎に住もう！と思っても、どこに住んだらいいのかわからない。縁のある土地は無いし、そもそもどんな田舎に住みたいんだろう？どの程度田舎で、どの程度都会に住みたいんだろう？まずはそこから考えてみよう。

<!--more-->
<h2>住む田舎に欲しい田舎要素</h2>
まずは田舎要素を挙げてみる。といっても正確に言うと都会に住みたくないのであって、不便さを求めているわけじゃないからちょっと書きづらいけど。
<ul>
 	<li>高い建物がない</li>
</ul>
むしろあったら都会と言ってもいいかもしれない。
<ul>
 	<li>建物がひしめき合ってない</li>
</ul>
ふわっとしてるなー。ひしめいているとそれは都会。
<ul>
 	<li>空気がきれい</li>
</ul>
さっきよりさらにふわっとしてきている気がするぞ？

やっぱり書きづらい…。逆に、最低限欲しい都会要素を探そうか。
<h2>住む田舎に欲しい都会要素</h2>
<ul>
 	<li>インターネットが使える/携帯の電波が届く</li>
</ul>
さすがにこのくらいは無いと、できる仕事にもかなり制約が出てきそう。さすがに両方使えない生活は今のところなしかなー。
<ul>
 	<li>Amazonが届く</li>
</ul>
これもとっても大事。現代っ子なので通販使えないと辛い。当日配達とか贅沢は言わないので。そもそもAmazonが届かないエリアってどのレベルだ？
<ul>
 	<li>東京に日帰りできる</li>
</ul>
これは優先度はそこまでは高くないけど、このくらいは盛り込まないと、逆に絞れなそうかな、と。友人と気軽に会えるくらいの距離がいいですねー。

うんうん、こっちのほうが書きやすいな。続けて逆の視点でも考えてみよう。
<h2>住む田舎に必要ない都会要素</h2>
<ul>
 	<li>近所にコンビニ</li>
</ul>
あれば便利だけど、なくても問題ない。そもそもコンビニ自体無いようなところでも問題ない気がする。買い物は日中スーパーに車で15分とか。30分レベルになるとちょっとつらいかな...
<ul>
 	<li>頻繁な電車</li>
</ul>
全くない、となると色々不便な気がするので、まあ1時間に1本くらいあるといいなー。（これってなかなか田舎度合い測るのに良い指標だったりしないだろうか。）基本車社会のところでいいのではないでしょうか。
<ul>
 	<li>テレビ放映</li>
</ul>
できればtvkとTOKYO MXが映ってほしいけど、そんな贅沢言ってたら田舎に住めないので、アニメはネットで見ます。となると、写ってほしいチャンネルも特になし。

都会基準だと色々出てくるなー。ただ、書いてて思ったけど、そもそもなんで都会を出たいんだろう。
<h2>都会の嫌いなところ</h2>
じゃあ都会を出たい理由を挙げてみよう。そもそもどうして田舎に住みたいと思ったんだろう。
<ul>
 	<li>人が多い</li>
</ul>
うん、確かに人が多い。
<ul>
 	<li>満員電車がやだ</li>
</ul>
うん、確かに嫌だ。
<ul>
 	<li>コンクリートジャングルは暑い</li>
</ul>
うん、確かに暑い。
<ul>
 	<li>空気がまずい</li>
</ul>
うん、確かにまずい。

あれ？なんか大した理由がないな。これ、実は家から出なければおおむね田舎でも都会でも問題ないんじゃないだろうか。空気だけはまずいけど。田舎は不便だし虫も多いし、いろいろ不便じゃないかな。そしたら別に田舎住むこともないのでは？
<h2>結局のところ</h2>
どうなんだろう。田舎に住みたいのかどうか、自信がなくなってきた。でも憧れはたしかにある。ひぐらし、のんのんびより、ばらかもん、サクラクエスト、おおかみこども、田舎が舞台の作品なんかを見ると、心躍るものがあるのは確かだ。けど、本当に都会を出ていくだけのものがあるのかな。良くない部分と良い部分を合わせて、プラスになるのかな。田舎に住んだことない今の自分にはきっと測り切れないので、結局のところやってみないとわからないことだらけだ。
